Web前端开发

Web前端开发是创建网页或APP界面呈现给用户的过程。Web前端开发技术主要包括HTML , CSS及JavaScript、JQuery以及 衍生出来的各种技术、框架。 本课程由高校与企业联合打造,将坚实的基础、实用的案例以及企业级的应用贯通交融,可以作为计算机与相关专业的Web前端开发入门课程。

加入学习

HTML

HTML可以设定网页的结构,有那些内容组成。HTML是用来描述网页的一种语言。HTML指的是超文本标记语言(Hyper Text Markup Language) HTML不是一种编程语言，而是一种标记语言(markup language) 标记语言是一套标记标签 (markup tag)HTML使用标记标签来描述网页

CSS

CSS指层叠样式表(Cascading StyleSheets)样式定义如何显示HTML元素样式通常存储在样式表中把样式添加到HTML 4.0中, 是为了解决内容与表现分离的问题外部样式表可以极大提高工作效率外部样式表通常存储在CSS文件中多个样式定义可层叠为一。CSS可以设置页面样式，做到样式与内容分离

开始学习

JavaScript

JavaScnipt是世界上最流行的脚本语言。JavaScript是属于web的语言,它适用于PC、笔记本电脑、 平板电脑和移动电话。JavaScript被设计为向HTML页面增加交互性。 许多HTML开发者都不是程序员，但是JavaScnipt却拥有非常筍单的语法。几乎每个人都有能力将小的JavaScript片段添加到网页中。

Jquery

jQuery是一个JavaScipt函数库。jQuery库包含以下特性: HTML元素选取。HTML元素操作。CSS 操作。HTML 事件函数。JavaScript特效和动画。HTML DOM遍历和修改。

开始学习

HTML5

HTML5是最新的HTML标准。HTML5是专门为承载丰富的web内容而设计的,并且无需额外插件。HTML5 拥有新的语义、图形以及多媒体元素。 HTML5 提供的新元素和新的API简化了web应用程序的搭建. HTML5是跨平台的,被设计为在不同类型的硬件(PC、平板、手机、电视机等等)之上运行。
